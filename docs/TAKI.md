## やること

### セットアップ
- [ ] Xcode で `ios-app/` のプロジェクトを開いて実機ビルドできる
- [ ] 実機の「デベロッパを信頼」設定済み

### センサー（CoreMotion）
- [ ] 加速度・傾きの値を取得できる
- [ ] 「腰（刀）」の構え判定
- [ ] 「肩の後ろ（大剣）」の構え判定
- [ ] 抜刀完了（引き抜きの移動量）の判定ロジック

### 振動（CoreHaptics）
- [ ] 振動を 1 発鳴らす（★初手）
- [ ] 刀：抜刀の「ジャキィン」
- [ ] 大剣：重い減衰

### スマホUI（SwiftUI ／ みずきから移管）
- [ ] スタート画面
- [ ] 剣セレクト画面
- [ ] 抜刀待機画面
- [ ] UI とセンサー・振動ロジックの結線（画面遷移・状態受け渡し）

### 連携（はると協働）
- [ ] 抜刀完了・構え完了のイベント／状態を、はるの Firestore 連携に渡せる形で公開
- [ ] はると組んで、抜刀検知 → `players/p1/drawn = true` の繋ぎ込みテスト

## ★ 初手のアクション
1. Xcode プロジェクトを立ち上げ、iPhone 実機を繋ぐ
2. **CoreHaptics で簡単な振動を 1 発**鳴らせるか確認
3. 続けて **SwiftUI で画面ラフを 1 枚**作ってみる
